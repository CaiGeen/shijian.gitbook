# 关于本书

### 概述

内容简介：分享如何使用TimeTrack/aTimeLogger软件记录时间，提高时间感知度，进入“记录-归类-分析-改变”正向循环，主动控制时间流向，最终达成目标的方法。

数据说明：本书中**示例均为脱敏数据**，不具有现实意义，仅供参考。

全文字数：六章，53节，5万字

软件环境：TimeTrack（[常见问题答疑](https://shijian.tujunjie.com/ch06/ch06.47)），iOS 12，Windows 10
